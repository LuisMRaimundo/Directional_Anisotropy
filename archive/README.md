# Archive — not part of the active pipeline

Files here are **historical or obsolete exports**. They are **not** used by:

- `run_analysis()` / `anisotropia/pipeline.py`
- `Anisotropia.py` (Streamlit)
- `pytest tests/`
- `corpus/scripts/*`
- GitHub Actions (`.github/workflows/tests.yml`)

## Layout

| Folder | Contents |
|--------|----------|
| `docs_legacy/` | Old StackEdit `.mhtml` snapshots of manuals (superseded by root `*.md`) |
| `scripts_legacy/` | Reserved for retired one-off scripts |
| `outputs_legacy/` | Reserved for obsolete local outputs |
| `notes_legacy/` | Reserved for draft notes |

**Canonical documentation** lives at the project root and in `docs/` — see `README.md`.
